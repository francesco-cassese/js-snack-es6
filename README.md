# JS Snack Array e Oggetti

## 📌 Descrizione
Una raccolta di esercizi focalizzati sull'integrazione tra array e oggetti. L'obiettivo è imparare a gestire strutture dati complesse, estrarre informazioni specifiche e trasformare collezioni di oggetti in nuovi formati.

## 🎯 Obiettivi

### 1. Racing Bikes (Ricerca del Minimo)
Gestione di un set di dati relativo a biciclette da corsa per individuare l'eccellenza in termini di leggerezza.
- **Focus Tecnico:** Iterazione di un array di oggetti e confronto di una proprietà numerica (`peso`) per isolare l'elemento con il valore minore.
- **Output:** Visualizzazione dinamica dei dati della bici più leggera.

### 2. Football Teams (Generazione e Trasformazione)
Simulazione di una classifica calcistica con generazione di statistiche casuali e filtraggio delle proprietà.
- **Focus Tecnico:** - **Randomization:** Popolamento dinamico delle proprietà `punti fatti` e `falli subiti`.
    - **Data Mapping:** Creazione di un nuovo array derivato che contenga solo un set ridotto di informazioni (nome e falli subiti), mantenendo l'integrità dei dati originali.
- **Output:** Stampa in console della nuova struttura dati generata.

## 🛠️ Tecnologie Utilizzate
- **HTML5:** File di supporto per l'esecuzione degli script.
- **JavaScript (ES6+):** Utilizzo di oggetti letterali, array di oggetti, cicli `for` e generazione di numeri casuali con `Math.random`.

---

## 🧪 Flusso di Lavoro
1. **Definizione:** Creazione delle strutture dati iniziali (array di oggetti).
2. **Elaborazione:** Scansione degli array per calcolare minimi o aggiornare valori.
3. **Trasformazione:** Estrazione dei dati necessari per la creazione di nuovi array di output.
4. **Verifica:** Controllo dei risultati tramite `console.log`.
